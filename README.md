# 🍽️ Маси — Ресторант

Уеб приложение за управление на маси и поръчки в ресторант. Работи на телефон,
запазва информацията и може да се добави като икона на началния екран.

## Маси

- **Салон:** маси 1–14
- **Градина:** маси 21–28
- **Тераса:** маси 31–38

## Какво прави

- Въвеждаш поръчки на всяка маса (всеки ред е отделно ястие/напитка).
- Всеки ред се отмята щом го занесеш (✓).
- Бутон **„Занесох всичко"** отмята всички наведнъж.
- Бутон **„Изчисти масата"** маха всичко и масата става свободна.
- Цветове на масите:
  - 🟢 **Зелено** — свободна
  - 🔴 **Червено и премигва** — чака ястия (има неотметнати поръчки)
  - 🟠 **Оранжево** — заета (всичко е занесено)
- Информацията се пази на устройството (в браузъра), затова **не изчезва**,
  като затвориш приложението.

## Как да го пуснеш и да си сложиш пряк път

Приложението се публикува автоматично през GitHub Pages.

### 1. Включи GitHub Pages (еднократно)

1. Отвори хранилището в GitHub → **Settings** → **Pages**.
2. При **Build and deployment → Source** избери **GitHub Actions**.
3. Готово. След първото качване ще получиш линк от вида:
   `https://kolyo17.github.io/resturant-project/`

> Линкът се вижда и в раздел **Actions → Deploy to GitHub Pages** след
> успешно изпълнение.

### 2. Добави пряк път на телефона

**iPhone (Safari):**
1. Отвори линка в Safari.
2. Натисни бутона за споделяне (квадратчето със стрелка нагоре).
3. Избери **„Към началния екран" / „Add to Home Screen"**.
4. Иконата се появява на екрана и се отваря на цял екран, без адресна лента.

**Android (Chrome):**
1. Отвори линка в Chrome.
2. Меню (трите точки) → **„Добавяне към началния екран"**.

## Технически бележки

- Чист HTML/CSS/JS, без зависимости. Целият код е в `index.html`.
- `manifest.json` + `sw.js` правят приложението инсталируемо и офлайн.
- Зумът в Safari е спрян нарочно (фиксиран мащаб + полета с шрифт ≥16px),
  за да не „скача" екранът при докосване.
- Данните се пазят в `localStorage` на устройството под ключ
  `restaurant_tables_v1`.
